# Carnap-MSU

Carnap-MSU is a fork of [Carnap](https://github.com/Carnap/Carnap). Carnap-MSU (like Carnap) is a free and open-source Haskell framework. Carnap-MSU runs [Carnap: Mississippi State](https://validit.ee/), a website for learning and teaching logic. 

See the [Carnap documentation](https://github.com/Carnap/Carnap#carnap) for a description of the project.
